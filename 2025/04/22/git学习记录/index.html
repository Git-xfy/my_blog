4. git中文件的四种状态

Untracked（未跟踪）
–文件刚被创建，还没有被 Git 认知
Modified（已修改，未暂存）
–文件之前已经被提交（Committed）过，当前在工作区里被修改过，但还没用 git add 放到暂存区。
Staged（已暂存，待提交）
–文件的当前版本已被 git add，放入暂存区（Staging Area），等待下一次 git commit。
Committed（已提交）
–文件的某个版本已经被记录在本地仓库（Local Repository）中。
